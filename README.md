# CodeAlpha_Basic-Chatbot
This is a simple rule-based chatbot created using Python. The chatbot responds to user input using a predefined dictionary of questions and answers. Users can type messages such as "hello", "how are you", "who", or "bye" to interact with the bot. Each input is converted to lowercase to ensure consistent matching. If the user input exists in the dictionary, the chatbot returns the corresponding response. If the input is not recognized, the bot replies with "I don't understand!". The conversation continues in a loop, allowing continuous interaction with the bot. The program stops when the user types "bye", at which point the bot says goodbye and exits. This project demonstrates the use of dictionaries, loops, conditionals, and basic input/output in Python. It is a beginner-friendly project for understanding how simple chatbots work.
